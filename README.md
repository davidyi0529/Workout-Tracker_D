## Installation

<br />

1. To run the application locally, first clone this repository with the following command.

<br />

```bash
git clone https://github.com/davidyi0529/Workout-Tracker_D.git
```

<br />

2. Next, install the application dependencies inside the Workout-Tracker_D directory.

<br />

```bash
npm install
```

<br />

3. Start local host server with server.js.

<br />

```bash
node server.js or npm start
```

<br />

4. Navigate to localhost:3000 in the browser.

<br />

5. Now you can log in your workouts.
